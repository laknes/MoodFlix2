# Moodflix - AI Movie Intelligence

Moodflix is a sophisticated, mood-based movie recommendation engine that uses Gemini AI to analyze your emotional layers and provide the perfect cinematic match.

## 🚀 Quick Start (Installation)

We have provided a comprehensive shell script to handle installation, updates, and SSL configuration.

### Deployment Script: `install.sh`

The `install.sh` script automates the entire process of setting up Moodflix on a Linux server.

#### Features:
1.  **Full Installation**: Sets up Nginx reverse proxy, prompts for database credentials, and configures admin panel defaults.
2.  **Seamless Updates**: Updates the site from Git without affecting your database or existing configurations.
3.  **SSL Management**: Built-in support for Let's Encrypt (Certbot) and Cloudflare Origin CA.

### How to use:

1.  **Upload the script** to your server root.
2.  **Give it execution permissions**:
    ```bash
    chmod +x install.sh
    ```
3.  **Run the script as root**:
    ```bash
    sudo ./install.sh
    ```

## 🔑 Environment Variables

The application requires a Google Gemini API Key. This can be configured in the Admin Panel or via the `.env` file generated by the installation script.

```env
API_KEY=your_gemini_api_key_here
PORT=3000
```
